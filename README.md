# Haag Family Missionary Map

Upload these files to your GitHub Pages repository:

- index.html
- style.css
- app.js

Your Google Sheet must be published to the web and use these tab names:

- Form Responses 1
- Coordinates

The Coordinates tab should include:

- Official Mission name (Ex: Maryland Baltimore)
- Latitude
- Longitude

The map matches missionary rows to coordinates by official mission name.
